Design Rationale

Sanisha Sapkota

Workflow Logic

Dashboard is structured around the VINTARA sales workflow: lead comes in via enquiry form → contacted → order placed → shipped → delivered. The pipeline view makes this visible at a glance, reducing missed follow-ups.

Usability Choices

Sidebar navigation for quick access to all CRM sections. Sticky topbar with search for finding clients instantly. One-click "Follow up" and task check buttons minimize admin effort. Toast notifications confirm every action.

Real-world Agency Use

Enquiry list is linked to sanishacore@email.com — same email used in Task 1 enquiry form, creating a seamless lead-to-CRM pipeline. Revenue chart and top products help Sanisha decide which products to restock or promote.
